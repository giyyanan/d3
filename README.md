# d3

Small D3 hierarchy and sunburst visualization repo.
The presence of flare.json, data.json, links, and sunburst suggests this was a visualization playground for hierarchical JSON data rather than a full app.

## What it was trying to do
- Experiment with hierarchical D3 visualizations.
- Use sample JSON data to drive a sunburst-style layout.
- Keep a reference link to the upstream flare dataset.

## Main parts
- sunburst - visualization data/config payload
- data.json and flare.json - sample hierarchical data
- links - reference URL for the dataset source
